Langages de Programmation

  • HTML5 - Niveau avancé
    Création de pages web structurées avec les balises sémantiques, formulaires avancés, multimédia
  • CSS3 - Niveau avancé
    Mise en forme avancée, animations, responsive design avec media queries, flexbox et grid
  • JavaScript - Niveau avancé
    Programmation côté client, manipulation du DOM, gestion d'événements, AJAX, programmation asynchrone
  • PHP - Niveau intermédiaire
    Développement côté serveur, gestion de sessions, interaction avec bases de données, création d'APIs
  • Java - Niveau intermédiaire
    Programmation orientée objet, applications desktop, développement d'applications robustes
  • C - Niveau intermédiaire
    Programmation système, gestion de la mémoire, algorithmes, structures de données

  • SQL - Niveau intermédiaire

  • MySQL / MariaDB - Niveau intermédiaire

Services Web et APIs
