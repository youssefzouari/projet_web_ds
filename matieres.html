Mini-Projet Programmation Web 1

Titre : Site Web Personnel

Date : Janvier 2026

Description :

Création d'un site web personnel complet comportant 10 pages différentes présentant mon parcours académique, mes compétences, mes expériences et mes passions.

Technologies utilisées :

  • HTML5 avec balises sémantiques
  • CSS3 pour la mise en forme
  • JavaScript pour l'interactivité (quizz, validation de formulaires)

Fonctionnalités principales :

  • Navigation intuitive entre les pages
  • Quizz interactif avec correction automatique
  • Formulaire de contact avec validation
  • Design responsive
  • Tableaux de données
  • Galeries d'images
